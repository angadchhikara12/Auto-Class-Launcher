# Privacy Policy

Auto Class Launcher respects user privacy and is designed to operate locally on the user's device.

## Information Accessed

The application requests read-only access to Google Calendar using the following OAuth scope:

```text
https://www.googleapis.com/auth/calendar.readonly
```

This access is used exclusively to:

* read upcoming calendar events
* detect scheduled online classes
* automatically open Engageli meeting links

## Data Storage

The application stores OAuth authentication tokens locally on the user's device to maintain Google Calendar access between sessions.

No calendar data is uploaded, transmitted, sold, or shared with external servers or third parties.

## Data Collection

Auto Class Launcher does not:

* collect personal information
* collect passwords
* collect analytics
* track users
* sell user data

## Third-Party Services

The application uses:

* Google Calendar API
* Google OAuth authentication

Users are also subject to Google's privacy policies and terms when using Google services.

## Security

All authentication is handled directly through Google's official OAuth flow in the user's default web browser.
